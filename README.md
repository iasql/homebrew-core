# Alan Technologies, Inc - Homebrew formulae
[Homebrew](https://brew.sh/) formulae for _alantech_ packages.

## Usage
Add new tap:
```bash
brew tap alantech/homebrew-core
```

Install:
```bash
brew install <formula>
```

Update:
```bash
brew update
brew upgrade <formula>
```
